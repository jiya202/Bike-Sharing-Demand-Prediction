# Bike Sharing Demand Prediction

## Objective
To predict bike rental demand using machine learning.

## Dataset Used
Bike Sharing Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Model Used
Random Forest Regressor

## Conclusion
The model was trained to predict bike rental demand using weather and seasonal information. The project demonstrates the use of machine learning for demand forecasting.
